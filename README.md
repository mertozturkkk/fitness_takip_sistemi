🏋️‍♂️ Fitness Takip Sistemi
📌 Proje Hakkında
Fitness Takip Sistemi, bireylerin fiziksel aktivitelerini, sağlık verilerini ve kişisel hedeflerini sistematik bir şekilde yönetmelerini sağlayan bir yazılım uygulamasıdır.
Bu proje, Python dili kullanılarak Nesne Tabanlı Programlama (OOP) yaklaşımı ile geliştirilmiştir.
🎯 Amaç
Fitness süreçlerini dijital ortamda takip etmek
Sağlık verilerini düzenli analiz etmek
Hedef bazlı ilerleme sunmak

🧮 Hesaplamalar
BMI (Vücut Kitle İndeksi)
BMI = kilo / (boy * boy)
Boy metre cinsine çevrilir.
Kalori Hesaplama
Kalori = katsayı × süre × kilo
